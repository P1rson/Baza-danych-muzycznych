# Baza-danych-muzycznych
Projekt tworzy bazę danych (MUZYCY)- albumów, gatunków i muzyków z użyciem SQLIte. 
Po uruchomieniu programu baza danych zostanie automatycznie utworzona, jeśli nie 
istnieje.
# Struktura bazy
Program automatycznie tworzy cztery tabele:

genres – gatunki muzyczne

artists – artyści 

albums – albumy powiązane z artystami 

tracks – utwory powiązane z albumami i 
gatunkami
